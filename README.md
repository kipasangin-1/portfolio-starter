# Portfolio Starter (HTML + CSS + JS)

Ini template portofolio **sederhana** yang bisa kamu modifikasi cepat.
Dibuat untuk pemula (bahasa Indonesia).

## Struktur
```
portfolio-starter/
├─ index.html
├─ styles.css
├─ script.js
├─ assets/
│  ├─ favicon.svg
│  └─ CV.pdf (opsional, ganti dengan punyamu)
├─ images/
│  └─ profil.svg
└─ projects/
   ├─ color-flipper/
   │  └─ index.html
   └─ profile-card/
      └─ index.html
```

## Cara pakai
1. Buka folder ini di VS Code.
2. Install ekstensi **Live Server** (Ritwick Dey).
3. Klik kanan `index.html` → **Open with Live Server**.
4. Edit teks `NAMA KAMU`, `EMAILKAMU@contoh.com`, `USERNAME/REPO`.

## Deploy cepat
- **GitHub Pages**: push repo → Settings → Pages → Deploy from branch → `main` → `/root`.
- **Netlify**: drag & drop folder ke app.netlify.com atau connect ke GitHub.
